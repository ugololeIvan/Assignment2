Users like mindless choices

The number of times users click to get from one part of the webpage to an-other is not all that important. What is most important is not to make the user think too much before every click. It’s also important to ensure that users don’t feel uncomfortable about the choice they have made. This discomfort usually arises as a result of the uncertainty generated during their clicks.

The links or button names used should be clear enough to allow the user to feel closer to the result with each click.

Give your web users the assistance they need

In some cases, you might not be able to avoid giving your users difficult choices but in the event, this happens, you should always give web users as much help as possible

Examples of guidance that work well are;

  • The smallest amount of information that will help your web user
  • Placing the content such that a user will find it when he or she needs
  • Make the format of choices good enough that the user cannot mis it
